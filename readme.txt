Вот такая скромная реализация. Программа печатает в консоль заданное количество паролей заданной длины(минимум 4). При этом, в пароле гарантированно будет тот тип символов, на который согласен пользователь (за исключением той ситуации, когда все символы определённой категории удалены пользователем вручную - тогда этот тип не будет использоваться вовсе). При этом паттерн у пароля полностью рандомный.
